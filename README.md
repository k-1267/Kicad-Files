# KiCad Projects (Nixie Tube Boards)

趣味の電子工作として作成した KiCad 基板設計データをまとめたリポジトリです。  
主にニキシー管関連の基板（駆動基板／マウント基板）になります。

---

## Projects

### 1) in-18_MountingBoard
手に入るもので最大のニキシー管 IN-18 のマウント基板です。
駆動基板と同サイズのため重ねて使用できます。

### 2) NixieTube_DrivingBoard
ニキシー管用の駆動基板です。
外部マイコンとは I2C で接続し、GPIOエキスパンダ MCP23018 で点灯用の制御信号を生成します。  
生成した信号はニキシー管専用ドライバ K155ID1 に入力し、ニキシー管の各カソード（数字）を選択して点灯します。  
IN-18 だけでなく、電流制限抵抗などを調整することで他の管にも流用できます。
